Overview

This is web service landing page for the JumpCloud Coding Exercise. In this exercise, I created a simple web service that accomplishes the following

  • Exposes an HTTP endpoint at the URL [ /manage_file] that accepts a JSON payload
  • Depending on the JSON payload, it will either download the sample text file or returns the contents of the text file & displays it in the browser
  • Ensures the service will return proper status codes for all requests

In addition, I will be adding the following enhancements to the service

  • Containerization via Docker
  • Automated Deployment
  • Logging, Metrics & Monitoring
  • Unit Testing via pytest
